# Gestionnaires


## Nouvelle association ? Vous devez la créer.

Dans votre tableau de bord (accessible via "Mon compte > Mon espace"), cliquez sur "CRÉER UNE ASSOCIATION".

Renseignez les 4 champs obligatoires suivants :

- Nom de l'association
- Acronyme ou sigle de l'association
- Adresse mail de l'association
- Établissement de rattachement


![Nouvelle association](/images/nouvelle-asso.png)


:::warning ATTENTION !
Il revient au président de chaque association, ou membres ayant obtenu délégation de présidence, de compléter tous les autres champs disponibles sur la fiche association.
:::

:::danger PUBLIQUE OU NON ?

Cocher la case "Association visible dans l'annuaire" permet de publier cette association dans l'annuaire.

Toutes les informations la concernant présentes sur cette fiche, ainsi que celles ajoutées ensuite par l'association, seront visibles publiquement.

:::


:::info SITE OU HORS SITE ?

Cocher la case "Les chartes de cette association sont à jour" permet de considérer d'emblée cette association comme Site.

Elle n'aura temporairement plus à soumettre la charte Site.

Il s'agit d'un moyen de donner immédiatement accès à la soumission de projets sollicitant un fonds pour lequel la signature de la charte Site est normalement requise, en l'occurence FSDIE ou IdEx.

⚠️ Attention ! La fiche association indiquera néanmoins que la charte Site n'a pas été signée.

À l'association de faire, dès que possible, une demande d'agrément ! 

Pour l'UM, aucune asso n'est considérée comme hors site.
:::

## Association déjà créée ? Vous pouvez la gérer.

Dans votre tableau de bord (accessible via "Mon compte > Mon espace"), cliquez sur "GÉRER LES ASSOCIATIONS".

![Associations créées](/images/asso-cree.png)

Cliquer sur le crayon correspondant à une association vous permet d'éditer celle-ci.

Vous pouvez renseigner ou modifier tous les champs de la fiche association, comme le ferait son président ou un membre ayant délégation de présidence. 

:::danger COMBIEN DE MEMBRES EN CAPACITÉ DE S'AFFILIER À UNE ASSOCIATION ?
Le nombre de membres par association sur la plateforme est limité à 4, par défaut.

Vous pouvez toutefois modifier cette restriction via le champ dédié.
![Nombre de membres](/images/nb-membre.png)

⚠️ Attention ! Le nombre indiqué ne peut être inférieur au nombre de membres déjà affiliés.
:::

En tant que gestionnaire établissement, vous pouvez surtout :

- désactiver ou réactiver une association ;

- la publier dans l'annuaire ou l'en dépublier ;

- la passer Site ou hors Site ;

- désactiver ou réactiver la soumission de projets, notamment en cas de bilan de projet déjà subventionné non rendu.

![Actions association](/images/action-asso.png)

:::info D'AUTRES ACTIONS ET/OU SUR PLUSIEURS ASSOCIATIONS ?

Sur la page listant les associations, après avoir coché celles sur lesquelles une action est à effectuer, vous pouvez choisir cette action dans un menu déroulant et cliquer sur confirmer :

- envoyer un mail ;
- exporter en CSV ;
- exporter au format Excel ;
- supprimer ;
- activer ;
- désactiver.

![Actions association](/images/action-asso2.png)
:::

:::warning ATTENTION !

Désactiver une association rend impossible :

- de nouvelles demandes d'affiliation ;
- une demande d'agrément ou un renouvellement d'agrément ;
- la soumission de nouveaux projets ;
- la publication de l'association dans l'annuaire.

La désactivation d'une association n'entraîne aucune perte de données.

⚠️ La suppression d'une association est, quant à elle, irréversible, d'où la présence d'une fenêtre pop-up de sécurité visant à vous faire confirmer votre action, non par un simple clic mais en renseignant au préalable le mot "association".

![Suppression association](/images/supp-asso.png)

:::
