# Membres d'associations, porteurs de projets individuels ou membres de commission

## 1. Avec un compte CAS

Vous souhaitez réinitialiser le mot de passe de votre compte CAS.

Vous devez vous référer à la procédure en vigueur dans votre établissement.

En cas de difficulté, vous pouvez contacter son support numérique.



## 2. Avec un compte externe

Vous souhaitez réinitialiser le mot de passe de votre compte externe.

Dans votre tableau de bord (accessible via "Mon compte > Mon espace"), cliquez sur "MODIFIER LES INFORMATIONS PERSONNELLES" puis "MON MOT DE PASSE".

![MDP](/images/motDePasse.png)

Renseignez votre mot de passe actuel puis saisissez deux fois votre nouveau mot de passe.


>
> [!WARNING] ATTENTION ! 
> Veillez, pour la sécurité de vos données, à choisir un mot de passe long et complexe (contenant minuscules, majuscules, chiffres et caractères spéciaux).
